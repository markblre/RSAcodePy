# RSAcodePy
This program allows the encoding and decoding of messages using RSA encryption.

## How to use
Simply place the files "RSAcode.py" and "mathsTools.py" in the same directory and run "RSAcode.py".

## Examples
Public key : E = 257, N = 1073  
Private key : D = 353
